# Architecture Decision Records

This section captures repo-local architecture proposals and decisions that are
specific to `dirac-bia`.

Use ADRs here for:

- repo-scoped proposals that affect the BIA documentation model
- monitored architecture directions that are still `Proposed`
- accepted repo-local decisions that should remain durable

Use `dirac-framework` instead when a decision becomes broader shared DIRAC
architecture guidance rather than a repo-local BIA concern.
